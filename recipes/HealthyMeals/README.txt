PureEra — Healthy Meals
========================================

This folder holds the downloadable PDF files for the "Healthy Meals" collection
(Clean & nourishing).

Place recipe PDFs directly inside this folder. Each filename must exactly
match the "pdf" field of the corresponding recipe object in /data.js
(case-sensitive), for example:

  QuinoaBuddhaBowl.pdf
  GrilledSalmonSalad.pdf
  AvocadoToast.pdf
  ZucchiniNoodles.pdf
  GreekYogurtParfait.pdf
  BakedSweetPotato.pdf
  ChickpeaSalad.pdf
  OvernightOats.pdf
  LentilSoup.pdf
  GrilledChickenBowl.pdf
  VeggieWrap.pdf
  GreenSmoothie.pdf
  MilletKhichdi.pdf

This collection currently expects 13 PDF file(s), listed above and
also in PDF-CHECKLIST.txt in this same folder.

Once a real PDF is added with the correct filename, the "Download PDF"
button on that recipe's card will work immediately — no code changes needed.
This placeholder file is safe to delete once every PDF above has been added.
